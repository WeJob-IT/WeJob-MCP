## [0.1.0] — 2026-07-14

### Added
- Initial public release of the WeJob Claude plugin.
- MCP connector configuration pointing to `https://mcp.wejob.ch/mcp` over HTTPS.
- `wejob` skill guiding Claude to call the WeJob MCP tools for public job,
  formation, and company data instead of inspecting local sources.
- Marketplace root and Claude-only plugin manifest.
